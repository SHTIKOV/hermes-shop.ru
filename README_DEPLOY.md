# ГЕРМЕС — deploy notes

Это статический сайт. Если будете публиковать на GitHub Pages:

1. Загрузите файлы в корень репозитория.
2. Убедитесь, что в корне есть `index.html`, `robots.txt`, `sitemap.xml`, `.nojekyll`.
3. После публикации замените `BASE_URL` в сгенерированных HTML на ваш реальный домен, если он нужен для canonical.
